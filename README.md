# Huawei SMS Client
Portable SMS Client to Manage Huawei Wifi/Router SMS using Desktop Computer
## Requirements
* Node.js
* Electron
* Electron-Packager

## Installation
```bash
git clone https://github.com/eru123/huawei-sms-client.git
cd huawei-sms-client
electron-packager ./ --electron-version=8.0.1
```
